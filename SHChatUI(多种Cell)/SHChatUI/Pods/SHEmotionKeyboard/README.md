# SHEmotionKeyboard
> 使用中如果有其他问题可以[`告诉我`](https://github.com/CCSH/SHEmotionKeyboard/issues/new)
# 使用 pod 导入
```objc
pod 'SHEmotionKeyboard'
```
# 表情键盘支持
- [x] 图文混排
- [x] 自定义表情
- [x] 系统表情
- [x] gif
- [x] 添加收藏
- [x] 最近表情
- [x] 字符串与富文本互相转换
