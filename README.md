# 两种风格聊天界面(QQ、微信)
# 消息格式
文本、语音、图片、位置、拍照、视频、名片、提示消息、红包、表情
# 功能
消息长按、头像长按、消息发送状态、消息重发、消息点击、语音播放与暂停
