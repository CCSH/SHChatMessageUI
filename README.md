# 两种风格聊天界面(QQ、微信)
>微信版本较新建议使用微信版本
**另外附上[`朋友圈界面代码`](https://github.com/CCSH/SHFriendTimeLineUI)**
# 消息格式
* 文本
* 语音
* 图片
* 位置
* 拍照
* 视频
* 名片
* 提示消息
* 红包
* 表情
* Gif
* 阅后即焚
# 功能
* 消息长按
* 头像长按
* 消息发送状态
* 消息重发
* 消息点击
* 语音播放与暂停
* 消息预览
* 微信版 输入框随着输入自增高
>我这里微信qq主要区别是工具栏样式与气泡样式，微信部分有时间会补充进去有几种消息格式没有添加
